# Cách init dự án

## 1. Backend API

1. Cài đặt project với typescript, expressjs, mongoosee/TypeORM sẵn.
2. Cấu hình cấu trúc dự án cho hệ thống Restfull API (Model, Controller, Service, Route)
3. Chuẩn bị các hàm, function helper, utils dùng chung nếu có
4. Nếu làm team --> Bắt buộc phải deploy Database lên production trước (thì mới làm chung được)


## 2. React Admin

1. Cài đặt project với cấu trúc theo chuẩn nào đó do team thống nhất
2. Tạo các component dự kiến chắc chắn sẽ tái sử dụng chung cho nhiều nơi.(Ví dụ: button, input, heading, message lỗi, thành công ...Nói chung các UI mà các thành viên ai cũng có thể sử dụng nhiều nơi)
3. Các thư viện mà team thống nhất áp dụng cho dự án (axios, zustand, react-query, react-hook-form ...)
3. Style CSS chung
